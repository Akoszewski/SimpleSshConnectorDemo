# Simple SSH Connector

Simple SSH Connector is an Android app I built to make working with my remote machines easier from a phone. It lets me connect to a server over SSH, open a terminal, and manage files in a shared remote folder without switching between multiple tools.

The app can generate its own SSH key or connect with a server password, upload and download files through SFTP, delete remote files, and keep a terminal session running while I move around the app. I also added a file upload option, which is useful when I want to quickly send something from the phone to my server.
